# react-native-sketch

A react-native &lt;Sketch /&gt; component to draw with touch events. (*Only for iOS for now*)

## Getting started

Install [rnpm](https://github.com/rnpm/rnpm) to make things easy:
```bash
$ npm i -g rnpm
```

Then, with rnpm installed:
```bash
$ rnpm install react-native-sketch
```
